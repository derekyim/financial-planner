## Completed Implementation

The RAG and RAGAS evaluation framework has been implemented as follows:

### Files Created

| File | Description |
|------|-------------|

### Tech Stack (matching reference notebooks)

- **Embeddings**: `text-embedding-3-small` (OpenAI)
- **Vector Store**: Qdrant (in-memory)
- **Text Splitter**: `RecursiveCharacterTextSplitter` (chunk_size=500, overlap=50)
- **Retrieval**: k=5 documents

### Strategic Guidance Agent

Added as a new sub-agent in `agents.py`:
- Routes via supervisor when user asks for business strategy advice
- Uses RAG to retrieve relevant context from knowledge base
- Cites sources in responses
- Connects advice to financial metrics
